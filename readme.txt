Фронтед-приложение автомата напитков.

Приложение работает локально, для запуска небходимо сначало запустить
backend-приложение находящиеся в репозитории: https://github.com/Oleg0392/JuiceAutomatMachine-BackendApp.git
Внутри репозитория так же имеется файл readme.txt для справки.

1.Открыть проект в Visual Studio Code.
2.Запустить проект командой: node main.js
3.В браузере перейте по адресу: http://localhost:3302/

Инструкция по автомату:

 Пользовательский интерфейс.
 - Интерфейс делится на 3 блока:
 -- 1: Блок внесения суммы и получения сдачи, находится с лева, в котором расположены:
 --- 	1: Дисплэй отображающий внесённую сумму
 --- 	2: Кнопка выдачи сдачи
 --- 	3: Кнопки монет (10,5,2,1)
 -- 2: Блок с напитками.
 --- 	1: Сетка с напитками
 -- 3: Блок с опциями
 --- 	1: Кнопка для перехода на страницу администратора

 - Для покупки напитка необходимо:
 -- 1: Внести с помощью кнопок монет сумму равную или большую цены желаемого напитка.
 -- 2: Выбрать напиток и нажать на кнопку 'Купить' находящиеся под названием выбранного напитка.
 -- 3: При наличии сдачи на кассе - нажать на кнопку 'Забрать сдачу' в блоке внесения суммы.
 
 Интерфейс администратора.
 - Так же делится на 3 блока:
 -- 1: Блок редактирования доступности монет:
 --- 	1: Кнопки монет
 -- 2: Блок с напитками
 ---	1: Сетка с напитками
 -- 3: Блок с опциями
 ---	1: Кнопа для сохранения изменений
 ---	2: Кнопка для возврата на страницу пользователя
 
 Для перехода на панель администратора необходимо кликнуть на кнопку 'Администрирование'
 в блоке опций пользовательского интерфейса и ввести ключ доступа к старнице в диалоговом
 окне. Ключ: 123q, передаётся как параметр в адресной строке. (как по заданию)
 
 Для блокировки определённой монеты необходимо кликнуть по нужной монете, после чего
 монета поменяет цвет. Тёмный тон - заблокирована, светлый - рабочая.
 
 Для редактирования напитков в ячейках напитков имеются конпки 'Изменить', 'Цена' и 'Ост'
 для изменения имени, цены и остатка напитков соответственно. Кнопа 'Блокировка' не рабочая,
 т.к. не несёт пользы (из-за количества, забыл удалить).
 
 Для сохранения всех внесённых изменений - в блоке опций имеется единая кнопка 'Сохранить'.
 Для возврата на страницу пользователя так же в блоке опций есть кнопка 'Вернуться'.
